# Netflix Data Analysis

## Project Overview
This project focuses on exploratory data analysis (EDA) of the Netflix dataset to understand trends, patterns, and insights related to movies and TV shows available on Netflix.  
The analysis is performed using Python and common data science libraries.

## Objectives
- Explore the structure of the Netflix dataset  
- Analyze the distribution of movies vs TV shows  
- Study content trends over time  
- Identify top genres, countries, and ratings  
- Gain insights into Netflix content strategy  

## Tools & Technologies
- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## Project Structure
netflix-data-analysis/  
├── Netflix.ipynb  
└── README.md  

## Key Insights
- Netflix releases more movies than TV shows  
- Content additions increased significantly after 2015  
- The United States has the highest content production  
- Drama and Comedy are among the most common genres  


## Dataset
The dataset includes information about Netflix movies and TV shows such as:
- Title  
- Type  
- Release year  
- Country  
- Rating  
- Duration
